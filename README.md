# Contacts App (React Hooks)

Contacts application made with react hooks.

## Features

The project includes
- [x] add contact form 
- [x] search by name
- [x] delete contact 
- [x] edit contact

## Available Scripts
- **Clone** Repository.

- Open project directory

``` bash
cd contacts-app-React-
```

- Install dependencies

``` bash
npm install
```

- Run application

``` bash
npm start
```

- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


### Deployment

The app is deployed on Github Pages:https://olgaaleshina.github.io/contacts-app-React-/
